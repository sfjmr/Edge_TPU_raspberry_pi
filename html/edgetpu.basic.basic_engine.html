edgetpu.basic.basic_engine¶

class edgetpu.basic.basic_engine.BasicEngine(model_path)¶

A general inference engine that can be extended for specific usecases.

Parameters:model_path – Path to the model you want to use.
RunInference(input)¶

Runs inference with given input.

Parameters:input – 1-D numpy.array. Flattened input tensor.
Returns:(latency, output_tensors). Latency is milliseconds in float while output_tensors is 1-D numpy.array. If there are multiple output tensors, it will be compressed into a 1-D array. You can use get_all_output_tensors_sizes, get_num_of_output_tensors and get_output_tensor_size to calculate the offset for each tensor.
get_all_output_tensors_sizes()¶

Gets sizes of output tensors.

Each output tenor is a 1-D numpy.array. This function returns the sizes of all Arrays.

Returns:Numpy.array represents the sizes of output tensors.
static get_driver_version()¶
get_inference_time()¶

Gets the latency of the last inference.

This can be used by higher level engines for debugging.

Returns:A float.
get_input_tensor_shape()¶

Gets shape of required input tensor.

For models trained for image  classification / detection, it’s always (1, width, height, channels). After flatten, the 1-D array with size width * height * channels is the required input for RunInference.

Returns:1-D numpy.array.
get_num_of_output_tensors()¶

Gets number of output tensors.

Returns:An integer representing number of output tensors.
get_output_tensor_size(tensor_index)¶

Gets size of specific output tensor.

Parameters:tensor_index – integer, the index of the output tensor.
Returns:An integer representing the size of the output tensor.

Gets output_tensors of last inference.

This can be used by higher level engines for debugging.

Returns:A numpy.array.
